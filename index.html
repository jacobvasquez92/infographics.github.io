The NIST Risk Management Framework

A 7-Step Process for Integrating Security

The RMF Lifecycle

Prepare

Define mission, boundary, and risk tolerance.

Categorize

Categorize system based on impact (C-I-A).

Select

Select baseline of security and privacy controls.

Implement

Implement the chosen controls and document.

Assess

Verify that controls are implemented correctly.

Authorize

Authorize the system for operation.

Step 7: Monitor

Continuously monitor controls for changes in security posture.

Resource Allocation

Implement and Monitor phases are the most resource-intensive.

Controls by Impact

High-impact systems require a larger set of controls.
